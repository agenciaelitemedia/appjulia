

# Plano: Atualizar Título do Projeto

## Objetivo
Alterar o título do projeto de "Lovable App" para "JulIA - Gerencie sua IA".

## Arquivo a Modificar

| Arquivo | Alteração |
|---------|-----------|
| `index.html` | Atualizar título e meta tags |

## Alterações Específicas

### `index.html`

1. **Tag `<title>`**: Alterar de "Lovable App" para "JulIA - Gerencie sua IA"

2. **Meta tag `og:title`**: Atualizar para "JulIA - Gerencie sua IA"

3. **Meta tag `description`**: Atualizar para uma descrição mais adequada ao projeto

```html
<!-- De -->
<title>Lovable App</title>
<meta name="description" content="Lovable Generated Project" />
<meta property="og:title" content="Lovable App" />
<meta property="og:description" content="Lovable Generated Project" />

<!-- Para -->
<title>JulIA - Gerencie sua IA</title>
<meta name="description" content="JulIA - Gerencie sua IA de atendimento" />
<meta property="og:title" content="JulIA - Gerencie sua IA" />
<meta property="og:description" content="JulIA - Gerencie sua IA de atendimento" />
```

## Impacto

- O título aparecerá na aba do navegador
- O título aparecerá nos compartilhamentos em redes sociais (Open Graph)
- Melhora a identidade da marca no projeto
